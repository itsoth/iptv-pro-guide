# IPTV Pro Guide — Agent Configuration

## Agents

- **build-agent** (primary) — Build, deploy, fix code
- **seo-agent** (subagent) — SEO, structured data, meta tags, hreflang
- **content-agent** (subagent) — Content strategy, translations, keywords
- **monitor-agent** (subagent) — Audits, performance, broken links

## Commands

- `/build` — Build the Next.js project
- `/audit` — Full site audit
- `/seo` — SEO analysis
- `/deploy` — Build and deploy

## Skills

- `seo` — Structured data, hreflang, meta tags
- `build` — Next.js build, config, errors
- `netlify` — Deployment, netlify.toml, CI/CD

## MCP Servers

- `playwright` — Browser automation for testing rendered pages

## Important

- Quit and restart opencode after changing `opencode.json` or any `.opencode/` file
